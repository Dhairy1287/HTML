Concept of WWW

  • It stands for World Wide Web.
  • It is basically a system if internet servers that support specially formatted documents.
  • It is a collection of text pages,videos,digital photographs and animations we can access over the internet.
  • User access the World Wide Web facilities via cilent called a web browser.

HTTP Protocol

  1. HTTP stands for Hypertext Transfer Protocol.
  2. HTTP is based on the cilent-server architectute model and a stateless request protocol.
  3. 404 File Not Found is a common HTTP status code.
